# Product Context - MTS_MultAgent

## Почему этот проект существует

### Боль корпоративного клиента
В крупных корпорациях типа МТС сотрудники ежедневно сталкиваются с проблемой:
- **Информационный хаос**: Данные разбросаны по разным системам (Jira, Excel, Confluence)
- **Ручная работа**: Сотрудники тратят 2-4 часа в день на сбор и анализ данных
- **Ошибки**: Ручная обработка приводит к неточностям и пропущенной информации
- **Задержки**: Отчеты готовятся долго, что замедляет принятие решений

### Целевая аудитория
1. **Project Managers** - нуждаются в быстрой аналитике по проектам
2. **Team Leads** - требуют сводную информацию по командам
3. **Аналитики** - должны обрабатывать большие объемы данных
4. **Руководители** - нуждаются в актуальных отчетах для принятия решений

## Как система должна работать

### Основной workflow
1. **Получение задачи**: Система получает описание задачи или проекта
2. **Сбор данных**: Автоматически собирает информацию из Jira и Excel
3. **Анализ**: Находит релевантный контекст и выделяет ключевые метрики
4. **Формирование отчета**: Создает структурированный отчет в Confluence
5. **Выводы**: предоставляет сравнительный анализ и рекомендации

### Требования к пользовательскому опыту
- **Простота**: Один клик для запуска анализа
- **Скорость**: Результат в течение минут, а не часов
- **Надежность**: Консистентные и точные результаты
- **Прозрачность**: Понятный процесс выполнения
- **Гибкость**: Возможность настройки под разные проекты

## Ключевые метрики успеха
- **Время экономии**: Сокращение времени на 80%
- **Точность**: Повышение точности анализа на 25%
- **Удовлетворенность**: Оценка пользователей 4.5/5
- **Внедрение**: Использование в 80% проектов

## Интеграционные требования
### Jira Integration
- Чтение задач и комментариев
- Получение протоколов совещаний
- Поиск по проектным ключам

### Excel Integration
- Чтение файлов с таблицами
- Поиск данных по запросам
- Обработка числовых и текстовых данных

### Confluence Integration
- Создание новых страниц
- Форматирование таблиц
- Добавление комментариев и описаний

## Ограничения и допущения
- **Доступ**: Только для авторизованных пользователей
- **Форматы**: Поддержка .xlsx, .xls файлов
- **Объем**: Ограничение на размер файлов (100MB)
- **Токены**: Требуются персональные токены доступа

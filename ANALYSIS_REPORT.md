## 🛠️ Инструменты для очистки

### Автоматический скрипт
Создан `scripts/cleanup_project.py` с возможностями:
- **Предпросмотр:** `python scripts/cleanup_project.py`
- **Выполнение:** `python scripts/cleanup_project.py --execute`
- **Backup:** Автоматическое сохранение информации об удаленных файлах
- **Safety:** Проверка возраста файлов (>7 дней) и исключение защищенных директорий

### Ручная очистка
Если предпочтете ручное управление:

```bash
# Временные файлы в корне
rm stage1_text_analysis.txt stage2_final_json.json

# Дублирующиеся отчеты
rm reports/baseline_agent_test_* reports/validation_report_*

# Временные cleaned протоколы
rm reports/daily/cleaned_*.txt

# Старая структура этапов
rm -rf reports/daily/stages/
```
